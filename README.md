# processador-fotonico

Simulação de um processador fotônico usando Perceval (Quandela).

Objetivo: implementar e experimentar arquiteturas de interferômetros multiporta e portas quânticas fotônicas, inspiradas nos artigos de referência abaixo.

Referências:

- "An Optimal Design for Universal Multiport Interferometers", arXiv:1603.08788
- "Quantum Fredkin and Toffoli gates on a versatile programmable silicon photonic chip", arXiv:2308.14881

Estrutura:
- `src/` - código do projeto
- `notebooks/` - experimentos e protótipos
- `tests/` - testes unitários
- `docs/` - documentação
